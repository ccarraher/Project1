How to compile and run:
    1. Open VSCode with Python already installed on your PC
    2. Install official Python extension
    3. Click View > Command Palette
    4. Type Python: Select Interpreter and choose your python environment
    5. Navigate to driver.py, right click, and select 'Run Python File in Terminal'
    6. Follow the instructions provided in the terminal
